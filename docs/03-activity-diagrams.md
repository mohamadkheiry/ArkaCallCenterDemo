# نمودارهای Activity — کال سنتر هوشمند آرکا

> جریان‌های کاریِ اصلیِ سامانه به‌صورت نمودار فعالیت (Activity Diagram).

## ۱. جریان تماس ورودی و پاسخ هوش مصنوعی

```mermaid
flowchart TD
    A["تماس ورودی به داخلی"] --> B["Asterisk: ساخت UUID (داخلی + تماس‌گیرنده)"]
    B --> C["اتصال AudioSocket به Realtime Worker"]
    C --> D{"تلفن هوشمندِ فعال یافت شد؟"}
    D -->|خیر| Z1["قطع تماس"]
    D -->|بله| E{"کاربر فعال است؟"}
    E -->|خیر| Z1
    E -->|بله| F{"سقف دقیقه پر شده؟"}
    F -->|بله| Z1
    F -->|خیر| G["اتصال به OpenAI Realtime + ارسال دستورها و پایگاه دانش"]
    G --> H["پخش پیام خوش‌آمد"]
    H --> I["دریافت صدای تماس‌گیرنده و رونویسی"]
    I --> J{"پاسخ در پایگاه دانش هست؟"}
    J -->|بله| K["پاسخ از پایگاه دانش"]
    J -->|خیر| L["پخش پیام fallback + ثبت «سوال بی‌پاسخ»"]
    K --> M{"تماس ادامه دارد؟"}
    L --> M
    M -->|بله| I
    M -->|خیر| N["ثبت رونوشت، ضبط، دقیقه و توکن"]
    N --> O["پایان"]
```

## ۲. ساخت تلفن هوشمند (Provisioning)

```mermaid
flowchart TD
    A["کاربر: دکمه ساخت تلفن هوشمند"] --> B{"پایگاه دانشِ تأییدشده؟"}
    B -->|خیر| E1["خطا: ابتدا پایگاه دانش"]
    B -->|بله| C{"پیام خوش‌آمد ثبت شده؟"}
    C -->|خیر| E2["خطا: ابتدا پیام خوش‌آمد"]
    C -->|بله| D["قفلِ سریالی (جلوگیری از رقابت تخصیص)"]
    D --> F["تخصیص داخلیِ آزاد"]
    F --> G["Provisioning روی ایزابل (PJSIP + reload)"]
    G --> H{"موفق؟"}
    H -->|خیر| E3["وضعیت: ناموفق"]
    H -->|بله| I["ذخیره داخلی + تولید صوت خوش‌آمد + پیامک اطلاع"]
    I --> J["تلفن هوشمند فعال شد"]
```

## ۳. ثبت و پخش سوالات بی‌پاسخ

```mermaid
flowchart TD
    A["هوش مصنوعی پاسخ fallback داد"] --> B["آخرین سوالِ تماس‌گیرنده به فهرستِ بی‌پاسخ افزوده می‌شود"]
    B --> C["پایان تماس: ذخیره در UnansweredQuestionsJson"]
    C --> D["کاربر: باز کردن بخش «سوالات بی‌پاسخ»"]
    D --> E["GET /api/calls/unanswered (تجمیع همه‌ی تماس‌ها)"]
    E --> F["کاربر روی «پخش» یک سوال کلیک می‌کند"]
    F --> G{"صوتِ کش‌شده موجود است؟"}
    G -->|بله| I["پخش mp3 کش‌شده"]
    G -->|خیر| H["تولید TTS با گوینده کاربر + ذخیره در کش"]
    H --> I
```
